# Jobvit Test

Next.jsとReactを使用したアプリケーションです。

## セットアップ

依存関係をインストールします：

```bash
npm install
```

## 開発サーバーの起動

開発モードでアプリケーションを起動します：

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて結果を確認できます。

## ビルド

本番用のビルドを作成します：

```bash
npm run build
```

## 本番サーバーの起動

ビルドしたアプリケーションを起動します：

```bash
npm start
```

## 技術スタック

- **Next.js** 14.2.5
- **React** 18.3.1
- **TypeScript** 5.5.4
